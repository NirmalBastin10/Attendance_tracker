# Attendance Tracker

**Attendance Tracker** is a simple web app to add subjects, mark attendance, and track attendance percentages.  
It stores all data using browser **localStorage**, so no backend is required.

## Features  
- Add and delete subjects  
- Mark present or absent  
- Edit attendance values  
- Auto-calculate attendance percentage  
- Shows “Today” and “Yesterday” for recent updates  
- Data saved using localStorage  

## Tech Stack  
- HTML  
- Tailwind CSS  
- JavaScript
